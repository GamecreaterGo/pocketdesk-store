PocketDesk — Product Images
═══════════════════════════

Place your keyboard photos in this folder using these exact filenames:

  keyboard1.jpg  — Main hero / card image (recommended: front view, clean background)
  keyboard2.jpg  — Folded view (showing compact size)
  keyboard3.jpg  — Open / in-use shot (paired with a phone or tablet)
  keyboard4.jpg  — Close-up of keys / texture detail
  keyboard5.jpg  — Lifestyle shot (on a desk, café, etc.)
  keyboard6.jpg  — Packaging / in-the-box shot

Recommended specs:
  Format:     JPG (or rename .png → update products.json image paths accordingly)
  Resolution: 1200 × 1000 px minimum (displayed at 600 × 500)
  Ratio:      4:3 landscape works best with the gallery layout
  File size:  Keep under 300KB each for fast loading
              (use squoosh.app or imageoptim.com to compress)

Once you've added the files, the website will display them automatically —
no code changes needed.
